# pomato
Potarble on messaging app tiny organizer

## Feature
- Posting a text
- Supporting Slack and LINE
- Built-in api token

## How to build
1. Get necessary package

    ```
    $ go get github.com/line/line-bot-sdk-go/linebot
    ```

1. Edit token.json file
1. Build go files
   
   ```
   $ go build
   ```

## How to use
```
$ pomato -m おやすみなさい -s -l
```
